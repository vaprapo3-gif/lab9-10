# 📘 Лабораторна робота

## 🔹 1. Підготовка середовища

1. Відкрити середовище LaTeX (наприклад, TeXmaker або Overleaf)  
2. Клонувати шаблон з GitHub Classroom у вашу робочу папку  

---

## 🔹 2. Ознайомлення з шаблоном

### Структура файлу
- Коментарі `% ...` показують, куди вставляти текст, формули та зображення  

### Підключені пакети
- `tcolorbox`  
- `amsmath`  
- `graphicx`  
- `fancyhdr`  

### Призначення пакетів
- **tcolorbox** — створення кольорових рамок (означення, теореми)  
- **amsmath** — запис математичних формул  
- **graphicx** — вставка зображень  
- **fancyhdr** — налаштування колонтитулів  

---

## 🔹 3. Введення тексту та формул

### Визначення, теореми, приклади
```latex
\begin{tcolorbox}[colback=blue!10!white, title=Визначення]
Квадрат — це чотирикутник з усіма сторонами рівними та всіма кутами прямими.
\end{tcolorbox}
```

### Список властивостей
```latex
\begin{enumerate}
    \item Усі сторони рівні.
    \item Усі кути прямі.
    \item Діагоналі перпендикулярні та рівні.
\end{enumerate}
```

### Доведення
```latex
\begin{proof}
Доведення властивостей квадрата...
\end{proof}
```

### Формули
```latex
$P_{ABCD} = 4AB$
```

---

## 🔹 4. Вставка зображень

### Основні правила
1. Зображення зберігайте в тій самій папці, що і `.tex` файл  

### Вставка зображення
```latex
\begin{figure}[h]
    \centering
    \includegraphics[scale=1]{foto2.png}
    \caption{Діагоналі квадрата}
    \label{fig:foto2}
\end{figure}
```

### Посилання в тексті
```latex
Як показано на рисунку \ref{fig:foto2}...
```

---

## 🔹 Як нумеруються зображення в LaTeX

- Зображення нумеруються автоматично (Рис. 1, Рис. 2, ...)  
- Нумерація відбувається у порядку появи в документі  
- Команда `\caption{}` додає підпис і номер  
- Команда `\label{}` створює мітку  
- Команда `\ref{}` дозволяє звертатися до номера рисунка
